# Slot-machine game using react native

![ScreenShot](https://github.com/SLYROOKO/Slot-Machine/blob/main/assets/ScreenShot.png?raw=true)

## Installation

```npm i```

## Development

```npm run start``
```npx run android```
```npx run-ios```

## Build

```npm run build-android-debug```
```npm run build-android-release```
